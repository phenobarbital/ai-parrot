# MCPServerRegistry

Defined in [`parrot.mcp.registry`](../summaries/mod:parrot.mcp.registry.md).

```python
class MCPServerRegistry
```

Catalog of pre-built MCP server helpers available for user activation.

Wraps the module-level ``_REGISTRY`` list with lookup and validation
methods.  The registry is declarative — all entries are hand-authored
above, not generated by reflection.

Example::

    registry = MCPServerRegistry()
    desc = registry.get_server("perplexity")
    params = registry.validate_params("perplexity", {"api_key": "sk-..."})

## Methods

- `def list_servers(self) -> List[MCPServerDescriptor]` — Return all registered MCP server descriptors.
- `def get_server(self, name: str) -> Optional[MCPServerDescriptor]` — Look up a single server descriptor by its registry slug.
- `def validate_params(self, name: str, params: Dict[str, Any]) -> Dict[str, Any]` — Validate user-supplied parameters against the descriptor schema.
