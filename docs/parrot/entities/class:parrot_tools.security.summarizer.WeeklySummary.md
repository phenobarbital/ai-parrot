# WeeklySummary

Defined in [`parrot_tools.security.summarizer`](../summaries/mod:parrot_tools.security.summarizer.md).

```python
class WeeklySummary(BaseModel)
```

A week-scoped security posture summary for one (provider, framework) pair.

Attributes:
    framework: Compliance framework (e.g., ``"HIPAA"``).
    provider: Cloud / infra provider (e.g., ``"aws"``, ``"container"``).
    period_start: Earliest ``produced_at`` timestamp among source scans.
    period_end: Latest ``produced_at`` timestamp among source scans.
    severity_totals: Element-wise sum of severity counts across all scans.
    new_findings: Findings present in current scans but NOT in the
        previous week's ``persistent_findings``.
    resolved_findings: Findings present in the previous week's
        ``persistent_findings`` but NOT in current scans.
    persistent_findings: Findings present in BOTH current scans and the
        previous week's ``persistent_findings``.
    executive_paragraph: 3–5 sentence narrative generated by the LLM.
    source_report_ids: UUIDs of the ``ReportRef`` scan reports consumed.
