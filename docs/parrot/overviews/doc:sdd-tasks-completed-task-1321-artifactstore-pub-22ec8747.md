## Context

Module 4 from the spec. The toolkit (TASK-1323) needs a way to expose the
frozen infographic HTML to an iframe / browser without forcing the client
through the session-scoped `GET /api/v1/artifacts/...` endpoint. Solution:
S3 sigv4 presigned URLs, generated by `ArtifactStore.get_public_url`,
delegating the signing call to the existing `OverflowStore`'s S3 client.

The resulting URL embeds nothing about the user (signature-only auth) and
has a max expiry of 7 days (S3 sigv4 hard limit).

- Add `async def get_public_url(...)` on `ArtifactStore`

- Add a helper on `OverflowStore` (`parrot/storage/overflow.py`) that wraps
  the underlying S3 client's `generate_presigned_url`. Name it
  `generate_presigned_url(key: str, *, expires_in: int = 604800) -> str`.
  Honour the 7-day cap (`min(expires_in, 604800)`).
- The `get_public_url` flow:
  1. Resolve the artifact's S3 object key (look up via the existing
     `get_artifact` + the overflow `ref` returned at save time).
  2. Call the new `OverflowStore.generate_presigned_url(...)`.
  3. Return the URL.
- Unit tests with a `botocore` stub (the codebase already uses `boto3`
  via the existing overflow store — re-use whatever mock pattern the
  existing storage tests use).

**NOT in scope**:
- The public HTTP route (`/api/v1/artifacts/public/...`) — that's TASK-1322.
- Any CSP / iframe handling — TASK-1322.
- A "permanent" non-expiring URL — S3 sigv4 caps at 7 days; documented.

## Implementation Notes

### Resolving the S3 key

The `OverflowStore.maybe_offload` already returns an `(inline, ref)` tuple
when persisting; that `ref` is what `Artifact.definition` carries when the
payload is offloaded. For frozen HTML produced by the toolkit (TASK-1323),
the artifact is saved with `definition.html` and the overflow ref is
returned synchronously at save time.

**Pattern** (sketch — adapt to the actual `ref` shape):

```python
async def get_public_url(
    self,
    user_id: Union[str, int],
    agent_id: str,
    session_id: str,
    artifact_id: str,
    *,
    format: Literal["html", "json"] = "html",
) -> str:
    artifact = await self.get_artifact(user_id, agent_id, session_id, artifact_id)
    if artifact is None:
        raise KeyError(f"Artifact {artifact_id} not found")

    # Derive S3 key from the overflow ref OR from the canonical key_prefix
    # used at save time. Use the same key_prefix logic as save_artifact
    # (look near line 55-57 to copy it verbatim — do NOT invent a new one).
    s3_key = self._resolve_key_for_format(artifact, format)

    return await self._overflow.generate_presigned_url(
        s3_key, expires_in=604800,
    )
```

```python
# parrot/storage/overflow.py
async def generate_presigned_url(self, key: str, *, expires_in: int = 604800) -> str:
    expires_in = min(int(expires_in), 604800)  # sigv4 hard cap
    # `self._s3_client` is the existing boto3 client used by maybe_offload.
    # boto3 client.generate_presigned_url is a sync call — wrap it via
    # `asyncio.get_event_loop().run_in_executor(None, ...)` to keep the
    # method async.
    loop = asyncio.get_event_loop()
    return await loop.run_in_executor(
        None,
        lambda: self._s3_client.generate_presigned_url(
            "get_object",
            Params={"Bucket": self._bucket, "Key": key},
            ExpiresIn=expires_in,
        ),
    )
```

### Key Constraints
- The signed URL MUST contain `X-Amz-Expires` ≤ `604800`.
- The signed URL MUST NOT carry `user_id` in the path or query.
- The method MUST be async.
- Honour the `format="html" | "json"` parameter: the JSON variant points
  to a different S3 object (the dynamodb-backed definition manifest); the
  HTML variant points to the offloaded HTML. If only one storage shape is
  meaningful in v1, raise `NotImplementedError` for the JSON variant and
  document the gap.
- Use `self.logger.info(...)` to log issuance events (no user_id in the
  log).

from parrot.storage.overflow import OverflowStore


@pytest.mark.asyncio
async def test_generate_presigned_url_clamps_expiry():
    store = OverflowStore(...)  # use existing test fixture

    url = await store.generate_presigned_url("k/path.html", expires_in=10_000_000)
    args, kwargs = store._s3_client.generate_presigned_url.call_args
