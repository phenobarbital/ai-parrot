# `parrot.knowledge.pageindex.schemas`

Pydantic models for PageIndex structured LLM outputs.

## Classes

- **`TocDetectionResult(BaseModel)`** — Result of checking whether a page contains a table of contents.
- **`TocCompletionCheck(BaseModel)`** — Result of checking whether a TOC extraction is complete.
- **`PageIndexDetection(BaseModel)`** — Result of detecting page numbers within a table of contents.
- **`TocItem(BaseModel)`** — A single entry in a parsed table of contents.
- **`TocJson(BaseModel)`** — Full table of contents in structured JSON form.
- **`TitleAppearanceCheck(BaseModel)`** — Result of checking if a section title appears on a specific page.
- **`TitleStartCheck(BaseModel)`** — Result of checking if a section starts at the beginning of a page.
- **`PhysicalIndexFix(BaseModel)`** — Result of finding the correct physical index for a section.
- **`GeneratedTocItem(BaseModel)`** — A TOC entry generated by LLM from document content (no-TOC mode).
- **`TreeSearchResult(BaseModel)`** — Result of an LLM tree search over a PageIndex tree.
- **`PageIndexNode(BaseModel)`** — A node in the PageIndex tree structure.
- **`PageIndexTree(BaseModel)`** — Top-level PageIndex document representation.
- **`DocDescription(BaseModel)`** — A one-sentence document description.
