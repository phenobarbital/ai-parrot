# AI-Parrot Pipelines

**ai-parrot-pipelines** provides specialized pipelines extracted from the AI-Parrot core package.

## Installation

```bash
pip install ai-parrot-pipelines
```

## Available Imports

```python
from parrot_pipelines.models import PlanogramConfig
from parrot_pipelines.planogram.plan import PlanogramCompliance
```

Backward-compatible imports from `parrot.pipelines` continue to work when `ai-parrot-pipelines` is installed.
