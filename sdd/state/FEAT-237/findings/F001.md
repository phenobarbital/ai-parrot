# F001: Embed method/return type

Base `EmbeddingModel` defines three methods: `embed_documents(texts) -> List[List[float]]`
(line 169), `embed_query(text) -> Any` (line 188), `encode(texts) -> np.ndarray` (line 226).

`SentenceTransformerModel.encode()` at huggingface.py:383 returns `np.ndarray` via
`loop.run_in_executor`.

For the PageIndex matmul use case, `encode()` is the right method -- returns numpy directly,
avoiding list->array conversion. `embed_documents` returns `List[List[float]]` after
`.tolist()` and Matryoshka truncation.
