# F006: EmbeddingRegistry architecture

Singleton at `EmbeddingRegistry.instance()`. CacheKey is `(model_name, model_type,
matryoshka_dim)` -- FEAT-150 extended to 3-tuple.

`get_or_create(model_name, model_type="huggingface", **kwargs)` async;
`get_or_create_sync()` sync variant.

`_build_model()` resolves via `supported_embeddings` map ->
`importlib.import_module(f"parrot.embeddings.{model_type}")`. kwargs flow through to
model constructor. Backend kwarg would flow through this path.
