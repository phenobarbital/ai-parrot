# F008: PageIndexRetriever LLM walk implementation

`search()` at line 38 serializes ENTIRE tree structure to JSON:
`tree_str = json.dumps(self.structure, indent=2)` (line 40). Sends it in a single LLM
prompt asking for relevant node_ids.

Confirms brainstorm's concern about scaling: prompt grows with tree size, and the entire
tree structure goes into one LLM call. The embedding beam walk (Phase B) would replace
this with O(depth) matmuls.
