# F002 — ai-parrot-tools layout & lazy TOOL_REGISTRY

## Summary

`ai-parrot-tools` ships under `packages/ai-parrot-tools/src/parrot_tools/`.
Tools are either single files (`bloomberg.py`, `fred_api.py`) or module
directories with a `tool.py` + `__init__.py` (`company_info/`, `ibisworld/`,
`odoo/`). Discovery is a lazy `TOOL_REGISTRY: dict[str, str]` in the package
`__init__.py` mapping a short name to a dotted import path. The registry is
auto-generated by `scripts/generate_tool_registry.py`, which **preserves
manual entries**, so a hand-added `"leadiq"` line survives regeneration.
